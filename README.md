# Math-Game-AI-Simulator-
Игровой движок и ИИ-агент для математической стратегии на Python. Спроектировал объектно-ориентированную архитектуру (ООП, наследование, полиморфизм). Реализовал алгоритм обратного поиска Minimax с мемоизацией состояний для оптимизации стратегий, а также алгоритм факторизации чисел для быстрой генерации ходов.
